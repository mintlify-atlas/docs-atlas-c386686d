Many RigidUI components have React Native variants, allowing you to build cross-platform applications with a consistent API.

## Available React Native Components

RigidUI provides React Native variants for the following components:

- **Image Loader** - Advanced image loading with gradient placeholders
- **File Uploader** - File picking and upload with progress tracking
- **Location Picker** - Location selection with native map integration
- **Infinite Scroll** - Efficient list rendering with infinite loading

## Installation

### Installing React Native Components

React Native components use the `-rn` suffix:

```bash
# Image Loader (React Native)
npx shadcn@latest add https://rigidui.com/r/image-loader-rn.json

# File Uploader (React Native)
npx shadcn@latest add https://rigidui.com/r/file-uploader-rn.json

# Location Picker (React Native)
npx shadcn@latest add https://rigidui.com/r/location-picker-rn.json

# Infinite Scroll (React Native)
npx shadcn@latest add https://rigidui.com/r/infinite-scroll-rn.json
```

### Additional Dependencies

React Native components require platform-specific dependencies:

#### Image Loader

```bash
npm install react-native-svg
```

#### File Uploader

```bash
npm install react-native-document-picker react-native-image-crop-picker
```

#### Location Picker

```bash
npm install react-native-maps @react-native-community/geolocation
```

#### Infinite Scroll

```bash
npm install @tanstack/react-virtual
```

## Platform Differences

### Styling

React Native components use React Native's StyleSheet API instead of Tailwind CSS:

#### Web Version

```tsx
import { cn } from "@/lib/utils"

function ImageLoader({ className, ...props }) {
  return (
    <div className={cn("relative overflow-hidden rounded-lg", className)}>
      <img {...props} />
    </div>
  )
}
```

#### React Native Version

```tsx
import { View, Image, StyleSheet } from "react-native"

function ImageLoader({ style, ...props }) {
  return (
    <View style={[styles.container, style]}>
      <Image {...props} />
    </View>
  )
}

const styles = StyleSheet.create({
  container: {
    position: 'relative',
    overflow: 'hidden',
    borderRadius: 8,
  },
})
```

### Component API

React Native components maintain a similar API to their web counterparts:

#### Web Component

```tsx
import { ImageLoader } from "@/components/ui/image-loader"

<ImageLoader
  src="https://example.com/image.jpg"
  alt="Description"
  className="w-full h-64"
  width={400}
  height={300}
/>
```

#### React Native Component

```tsx
import { ImageLoader } from "@/components/ui/image-loader-rn"

<ImageLoader
  src="https://example.com/image.jpg"
  alt="Description"
  style={{ width: '100%', height: 256 }}
  width={400}
  height={300}
/>
```

## Example: Image Loader

### Web Version

The web version uses SVG gradients and CSS animations:

```tsx
import { ImageLoader } from "@/components/ui/image-loader"

function ProductImage() {
  return (
    <ImageLoader
      src="/products/shoes.jpg"
      alt="Running shoes"
      className="w-full h-96 object-cover rounded-xl"
      width={800}
      height={600}
      loading="lazy"
    />
  )
}
```

### React Native Version

The React Native version uses `react-native-svg` for gradients:

```tsx
import { ImageLoader } from "@/components/ui/image-loader-rn"
import { View, StyleSheet } from "react-native"

function ProductImage() {
  return (
    <View style={styles.container}>
      <ImageLoader
        src="https://example.com/products/shoes.jpg"
        alt="Running shoes"
        style={styles.image}
        width={800}
        height={600}
      />
    </View>
  )
}

const styles = StyleSheet.create({
  container: {
    width: '100%',
    aspectRatio: 4 / 3,
  },
  image: {
    width: '100%',
    height: '100%',
    borderRadius: 12,
  },
})
```

### Implementation Details

The React Native version uses platform-specific gradient implementation:

```tsx
import Svg, { Defs, LinearGradient, Stop, Rect } from 'react-native-svg'

const GradientPattern = ({ colors }) => {
  return (
    <Svg width="100%" height="100%" style={StyleSheet.absoluteFillObject}>
      <Defs>
        <LinearGradient id="grad" x1="0%" y1="0%" x2="100%" y2="100%">
          <Stop offset="0%" stopColor={colors.start} stopOpacity="1" />
          <Stop offset="50%" stopColor={colors.middle} stopOpacity="1" />
          <Stop offset="100%" stopColor={colors.end} stopOpacity="1" />
        </LinearGradient>
      </Defs>
      <Rect width="100%" height="100%" fill="url(#grad)" />
    </Svg>
  )
}
```

## Example: File Uploader

### Web Version

```tsx
import { FileUploader } from "@/components/ui/file-uploader"

function DocumentUpload() {
  return (
    <FileUploader
      accept="image/*,application/pdf"
      maxSize={10 * 1024 * 1024} // 10MB
      onUpload={async (files) => {
        const formData = new FormData()
        files.forEach(file => formData.append('files', file))
        await fetch('/api/upload', { method: 'POST', body: formData })
      }}
      className="border-2 border-dashed rounded-xl p-8"
    />
  )
}
```

### React Native Version

```tsx
import { FileUploader } from "@/components/ui/file-uploader-rn"
import DocumentPicker from 'react-native-document-picker'

function DocumentUpload() {
  return (
    <FileUploader
      accept={[DocumentPicker.types.images, DocumentPicker.types.pdf]}
      maxSize={10 * 1024 * 1024} // 10MB
      onUpload={async (files) => {
        const formData = new FormData()
        files.forEach(file => {
          formData.append('files', {
            uri: file.uri,
            type: file.type,
            name: file.name,
          })
        })
        await fetch('https://api.example.com/upload', {
          method: 'POST',
          body: formData,
        })
      }}
      style={styles.uploader}
    />
  )
}
```

## Example: Infinite Scroll

### Web Version

```tsx
import { InfiniteScroll } from "@/components/ui/infinite-scroll"

function ProductList() {
  const loadMore = async (page: number) => {
    const response = await fetch(`/api/products?page=${page}`)
    return response.json()
  }

  return (
    <InfiniteScroll
      loadMore={loadMore}
      renderItem={(product) => <ProductCard {...product} />}
      className="grid grid-cols-1 md:grid-cols-3 gap-4"
    />
  )
}
```

### React Native Version

```tsx
import { InfiniteScroll } from "@/components/ui/infinite-scroll-rn"
import { View, StyleSheet } from "react-native"

function ProductList() {
  const loadMore = async (page: number) => {
    const response = await fetch(`https://api.example.com/products?page=${page}`)
    return response.json()
  }

  return (
    <InfiniteScroll
      loadMore={loadMore}
      renderItem={(product) => <ProductCard {...product} />}
      numColumns={2}
      style={styles.list}
    />
  )
}

const styles = StyleSheet.create({
  list: {
    padding: 16,
  },
})
```

## Utilities and Helpers

### cn() Utility

The `cn()` utility works differently on React Native:

#### Web (Tailwind CSS)

```tsx
import { cn } from "@/lib/utils"

const className = cn(
  "px-4 py-2 bg-primary",
  isActive && "bg-primary/90"
)
```

#### React Native (StyleSheet)

```tsx
import { StyleSheet } from "react-native"

const styles = StyleSheet.create({
  base: { paddingHorizontal: 16, paddingVertical: 8, backgroundColor: '#000' },
  active: { backgroundColor: '#333' },
})

const style = [styles.base, isActive && styles.active]
```

### Platform-Specific Code

Use React Native's Platform API for platform-specific behavior:

```tsx
import { Platform, StyleSheet } from "react-native"

const styles = StyleSheet.create({
  container: {
    padding: 16,
    ...Platform.select({
      ios: {
        shadowColor: '#000',
        shadowOffset: { width: 0, height: 2 },
        shadowOpacity: 0.1,
        shadowRadius: 4,
      },
      android: {
        elevation: 4,
      },
    }),
  },
})
```

## Best Practices

### 1. Use Platform-Specific Imports

Create separate files for web and native:

```
components/
  ui/
    image-loader.tsx      # Web version
    image-loader.native.tsx # React Native version
```

### 2. Responsive Design

Use React Native's Dimensions API:

```tsx
import { Dimensions, StyleSheet } from "react-native"

const { width, height } = Dimensions.get('window')

const styles = StyleSheet.create({
  container: {
    width: width * 0.9,
    maxWidth: 600,
  },
})
```

### 3. Handle Permissions

Request necessary permissions for location and file access:

```tsx
import { PermissionsAndroid, Platform } from "react-native"

const requestLocationPermission = async () => {
  if (Platform.OS === 'android') {
    const granted = await PermissionsAndroid.request(
      PermissionsAndroid.PERMISSIONS.ACCESS_FINE_LOCATION
    )
    return granted === PermissionsAndroid.RESULTS.GRANTED
  }
  return true
}
```

### 4. Optimize Performance

Use React Native's performance optimization techniques:

```tsx
import { memo } from 'react'
import { FlatList } from 'react-native'

const ProductItem = memo(({ item }) => <ProductCard {...item} />)

function ProductList({ products }) {
  return (
    <FlatList
      data={products}
      renderItem={({ item }) => <ProductItem item={item} />}
      keyExtractor={item => item.id}
      removeClippedSubviews
      maxToRenderPerBatch={10}
      updateCellsBatchingPeriod={50}
      windowSize={21}
    />
  )
}
```

## Limitations

### Components Not Available on React Native

Some components are web-only due to browser-specific APIs:

- **Access Manager** - Session management (use native auth solutions)
- **File Explorer** - File system browsing (use native file pickers)
- **Draggable Dashboard** - Complex drag-and-drop (use react-native-draggable)
- **Smart Form** - React Hook Form integration (use native form libraries)
- **Guided Tour** - DOM-based overlays (use native onboarding libraries)

### Styling Differences

React Native doesn't support:

- CSS pseudo-classes (`:hover`, `:focus`)
- CSS animations (use Animated API or Reanimated)
- Flexbox gaps (use margins instead)
- CSS Grid (use nested flex layouts)
