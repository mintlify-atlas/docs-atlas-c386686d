## What is RigidUI?

RigidUI is a collection of enterprise-grade UI components for React and Next.js applications, built on top of shadcn/ui. It extends the shadcn/ui ecosystem with additional complex components designed for real-world applications.

Instead of shipping components as a package, RigidUI follows the shadcn/ui philosophy: components are added directly to your project, giving you complete ownership and customization control.

<CardGroup cols={2}>
  <Card title="Registry-Based" icon="box">

  </Card>
  <Card title="Customizable" icon="palette">

  </Card>

## Component Categories

### Creative Components

Animated, interactive components that add visual polish to your applications:

- **Data Reveal** - Animated component that reveals sensitive content with smooth staggered animations and auto-hide functionality
- **Draggable Canvas** - Smooth, physics-based draggable canvas with momentum, elastic bounds, and customizable items
- **Text Shift** - Animated text component that smoothly transitions between two text strings with staggered letter animations
- **Creative Buttons** - Collection of beautiful, animated button components

### Core Components

Essential building blocks for complex applications:

- **Access Manager** - Authentication and authorization system with session validation
- **Currency Manager** - Multi-currency conversion and display system
- **Draggable Dashboard** - Interactive drag-and-drop dashboard with customizable widgets
- **Guided Tour** - Comprehensive guided tour component for interactive step-by-step walkthroughs
- **Image Loader** - Advanced image loading with loading states, error handling, and optimizations (Web + React Native)
- **Location Picker** - Geographic location selection with map integration and search (Web + React Native)
- **Smart Search** - Advanced search with real-time results, filters, history, and keyboard navigation

### Form Elements

Advanced form components for data collection:

- **File Uploader** - Drag and drop file upload with previews, validation, and progress tracking (Web + React Native)
- **Multi-Step Form Wrapper** - Guided step-by-step form interfaces with progress indicators
- **Password Strength Meter** - Visual password strength evaluation with customizable rules
- **Smart Form** - Zod-powered form with TanStack Query integration and automatic validation

### Data Display

Sophisticated components for presenting information:

- **Content Grid** - Flexible grid/list component with seamless view switching and customizable rendering
- **File Explorer** - Hierarchical file system browser with syntax highlighting and file operations
- **Infinite Scroll** - Efficiently load large datasets with infinite scrolling and virtual rendering (Web + React Native)
- **Notification Center** - Comprehensive notification management with categories and actions

### Hooks

Custom React hooks for enhanced functionality:

- **useLocation** - Custom hook for location-based functionality and geolocation services
